# Career Development Experience: Job Applications

## Adventure Description
For this assignment, I did a targeted application sprint for full-time technical roles and high-impact internships to prepare for my upcoming graduation. I completed 12 total applications, documenting the end-to-end process to build a comprehensive professional portfolio.

## Career Goals Context
My short term goal is to get a software engineering or database management position where I can immediately apply my experience in cross-platform development, relational databases, and object-oriented programming. My long-term goal is to lead technical projects and step into senior architectural roles within the tech industry, specifically focusing on creating scalable internal tools and reliable software systems.

## Deliverable Summary
This repository contains the documentation and proof of my application process. 
* **Application_Evidence Folder:** Contains screenshots of the job description, the completed application form, and the submission confirmation for each of the 12 roles.
* **Resume** My personal resume used for the applications.
* **Reflection.md:** A detailed report analyzing this career development exercise.
